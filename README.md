# uicms

A simple CMS made with laravel

# install

- composer install
or
- sudo composer install

# config db

- edit .env file

# migrations

- php artisan module:migrate

# seeds

- php artisan db:seed

# permissions

- chmod -R 777 storage
or 
- sudo chmod -R 777 storage

#run

- php artisan serve

- http://localhost:8000/uiadmin
